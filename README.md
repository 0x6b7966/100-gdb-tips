# 《100个gdb小技巧》

一本关于gdb使用小技巧的手册。100，在这里可能只是表明很多；具体的数目取决于您的参与和贡献。

## 如何参与

直接发PULL REQUEST。

增加一个小技巧的步骤：
1. 在src目录下新增一个md文件，参照现有文件的格式风格，编写一个小技巧
2. 在index.md中为新md文件增加一个索引，可以放到已有分类中，或增加一个分类
3. 如果预览下没有问题，OK!
